Principais formatações.

Negrito/ Destaque

Nesta frase Termo em negrito usando a tag B (não semântica)

Nesta frase tenho um termo em destaque usando a tag STRONG (semântica)

Itálico / Ênfase

Nesta frase temos um termo itálico usando a tag I (Não semântico)

Nesta frase temos um termo itálico usando a tag EM (Semântico)

Texto marcado

Podemos criar também um texto marcado usando a tag MARK.

Texto grande e pequeno.

Estamos criando um texto grande e um texto pequeno

Texto excluido/ Deletado

Podemos marcar um texto como excluído para indicar que ele deve ser lido, mas não considerado

Texto Inserido

Podemos marcar um texto com inserido para dar ênfase indicar que foi adicionado depois

Texto sobrescrito

Para inserir coisas do tipo x20+3 (conta matemática)

Texto subscrito

Para inserir coisas do tipo H2O (formula quimica)
